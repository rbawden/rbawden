DiaBLa

"Dialogue Bilingue"

Machine Translation-mediated bilingual dialogue corpus

Download the dataset Download the interface code
DiaBLa ("Dialogue Bilingue") is an English-French bilingual corpus of written dialogues mediated by machine translation systems. It is complete with human reference translations as well as fine-grained real-time judgments of translation quality produced by the users themselves. See this page for a description of the dataset. It is distributed under a CC-BY-SA-4.0 Licence.
